код на питоне для отображения температур пк
Для получения данных о температуре компьютера в Python можно использовать библиотеку PyWin32 и модуль WMI (Windows Management Instrumentation). Ниже приведен код для отображения температуры процессора и видеокарты:  ``` import wmi from time import sleep  # Создаем объект WMI и запрашиваем данные о температуре процессора и видеокарты w = wmi.WMI(namespace="root\\OpenHardwareMonitor") temperature_infos = w.Sensor()  # Отображение данных о температуре каждые 5 секунд while True:     for sensor in temperature_infos:         if sensor.SensorType == 'Temperature' and "CPU Package" in sensor.Name:             print("Температура процессора: %s C" % sensor.Value)         elif sensor.SensorType == 'Temperature' and "GPU Core" in sensor.Name:             print("Температура видеокарты: %s C" % sensor.Value)     sleep(5) ```  Обратите внимание, что установка и использование данной библиотеки требует прав администратора, а также необходимо убедиться в ее совместимости с вашей операционной системой. Кроме того, отображение температуры может отличаться в зависимости от производителя компьютера/компонентов.
